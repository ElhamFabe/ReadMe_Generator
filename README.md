  # Description  
  Creates an informative ReadMe.md file for any user.

  ### Technologies:
  Node.js
 ---

  ### Installation:
  - Fork repository
 - Install node.js
 - run app.js on integrated terminal (VSCode) or
 - run on terminal (macOS) or git bash (windows) (check to see if your are in the right path)
 - npm install (npm i)
 - run node index.js
